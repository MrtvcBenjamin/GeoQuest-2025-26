# Diplomarbeit_2025: GeoQuest - Die Schnitzeljagd App

Am Ende jedes Schuljahres organisiert die HTL Leoben eine Schnitzeljagd fuer die ersten Klassen. Bisher wurde dazu eine externe App genutzt, die jedoch nur eingeschraenkt anpassbar ist und Abhaengigkeiten von Drittanbietern schafft. Ziel ist daher die Entwicklung einer eigenen Anwendung, die flexibel erweiterbar, nachhaltig nutzbar und optimal an die schulischen Anforderungen angepasst ist.
